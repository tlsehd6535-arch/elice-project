# AI 에이전트 E2E 자동화 테스트 프로젝트 (qa3team)

이 프로젝트는 **elice-project**의 일환으로, AI 에이전트 서비스의 핵심 기능을 검증하기 위한 자동화 테스트 코드를 포함하고 있습니다.

## 주요 폴더 및 구성
* **/dongbin**: 동빈 담당 작업 영역
* **/dongbin_tests**: Pytest 기반의 테스트 스크립트 모음
* **/agent_scripts**: Selenium을 활용한 AI 에이전트 생성 및 대화 시나리오 자동화 로직

## 사용 기술
* **Framework**: Pytest
* **Library**: Selenium WebDriver
* **Language**: Python 3.x

## 주요 테스트 시나리오
1. AI 에이전트 생성 기능 검증
2. 에이전트와의 대화 기능 E2E 테스트
3. 예외 상황 및 핵심 비즈니스 로직 시나리오 자동화
